# Voice-Controlled-Smart-AI-Assistant
Python, Anthropic API, Speech Recognition, Web Scraping
Built an end-to-end voice assistant integrating Anthropic's Claude API with real-time speech-to-text (Google STT), text-to-speech (gTTS), and audio playback (pygame). Features push-to-talk recording, mid-sentence interrupt, multi-modal file analysis (PDF, images, text), and on-demand web page fetching with HTML parsing.
Practical Demonstration video link: https://www.youtube.com/watch?v=AaamF4kGLSM
